EyeSat - Capteur d'image CMOS
-----------------------------
 
### C'est quoi? 
EyeSat est un nano-satellite de 30x10x10cm réalisé par des étudiants.
Pour en savoir plus: [Blog du Spacelab de l'IUT de Cachan][1]
